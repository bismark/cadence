## Fixed

Added fixed width to the play/pause button:

```kotlin
EInkButton(
    onClick = onPlayPause,
    filled = true,
    modifier = Modifier.width(72.dp)  // Fixed width
) {
    Text(if (isPlaying) "Pause" else "Play")
}
```
