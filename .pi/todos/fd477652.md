# ZIP compression implemented for align command

The `align` command now properly creates ZIP bundles by default.

## Changes
- `writeBundleAligned()` now:
  1. Writes to a temp directory
  2. Creates ZIP archive with max compression (level 9)
  3. Cleans up temp directory

## Usage
```bash
# ZIP output (default)
cadence-compile align -e book.epub -a audiobook.m4b -o book.bundle.zip

# Uncompressed output
cadence-compile align -e book.epub -a audiobook.m4b -o book.bundle --no-zip
```
