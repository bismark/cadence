## Fixed

The image content format was wrong. Changed from Anthropic API format:
```javascript
{ type: "image", source: { type: "base64", media_type: "image/png", data: ... } }
```

To pi's expected format:
```javascript
{ type: "image", data: base64Data, mimeType: "image/png" }
```

File: `.pi/extensions/android-screenshot.ts`
