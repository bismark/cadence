## Goal
Create or collect additional test EPUBs that cover edge cases the compiler should handle.

## Edge Cases to Cover
- Very long paragraphs that span multiple pages
- Nested elements (spans inside spans, complex markup)
- RTL text (Arabic, Hebrew)
- Mixed LTR/RTL content
- Tables with synchronized audio
- Footnotes/endnotes
- Poetry/verse with line breaks
- Large books (1000+ spans)
- Multiple audio formats (MP3, M4A, OGG source)
- Missing/malformed SMIL files
- Unicode edge cases (emoji, combining characters)

## Approach
Could generate synthetic EPUBs programmatically to test specific scenarios.
