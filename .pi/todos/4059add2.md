## Goal
Automated test suite for the Android player.

## Test Categories
1. **Unit tests** - Bundle parsing, binary search, data classes
2. **Integration tests** - BundleLoader, AudioPlayer
3. **UI tests** - Compose testing, navigation, tap handling
4. **Instrumented tests** - On-device/emulator tests

## Test Cases
- Bundle loading and parsing
- Binary search correctness (findSpanAtTime)
- Span index lookups (getSpanById)
- Position persistence (PlaybackPreferences)
- Page navigation
- Playback state management
- Seek behavior

## Tools
- JUnit for unit tests
- Compose UI testing
- Espresso for instrumented tests
- CI integration (GitHub Actions with Android emulator)

## Files
- `player/app/src/test/` - Unit tests
- `player/app/src/androidTest/` - Instrumented tests
