# Problem
Pages show large empty whitespace because long paragraphs are forced to stay intact (no break inside), causing the browser to push the entire paragraph to the next column/page.

## Cause
`paginate.ts` injects:
```css
p, [data-span-id] { break-inside: avoid; }
```
This prevents column/page breaks inside paragraphs.

## Fix options
1. Allow breaks inside paragraphs; keep `break-inside: avoid` for spans only
2. Apply `break-inside: avoid` only to short blocks (headings) or spans

## Files
- `compiler/src/layout/paginate.ts` (column CSS injection)

## Test
Moby Dick page 23/79 shows big blank area after a paragraph.

- Updated column CSS to allow paragraph breaks; keep break-inside: avoid only on spans. (compiler/src/layout/paginate.ts)
